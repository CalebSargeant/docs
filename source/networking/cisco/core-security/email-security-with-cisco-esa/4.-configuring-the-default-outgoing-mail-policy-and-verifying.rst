--------------------------------------

-  Microsoft Server
-  MailEnable
-  AD Users and Computers, DNS, IIS

   PS C:\Users\ Administrator> Import-Module ActiveDirectory
   PS c:\Users Administrator> $User = Get-ADUser kinda -Properties proxyaddresses
