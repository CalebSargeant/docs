Download Videos
===============

https://github.com/ytdl-org/youtube-dl/blob/master/README.md

.. code-block:: bash

  # Install youtube-dl
  brew install youtube-dl

  # Example to extract and download mp3 from a YouTube playlist
  youtube-dl -i -x --audio-format mp3 https://www.youtube.com/playlist?list=PLAdhO41Xw-3WPuopzClryku_gtr5V7YXA

  # Example to download just a video
  youtube-dl -i https://www.youtube.com/watch?v=NZHn2UhrLZM
